分支与循环

if

单一的 if 语句

1if (条件) {
2	要执行的语句块;
3}

if else 语句

1if (条件) {
2	要执行的语句块;
3} else {
4	要执行的语句块;
5}

多重 if 语句嵌套

1if (条件) {
2	要执行的语句块;
3} else if (条件) {
4	要执行的语句块;
5} else {
6	要执行的语句块;
7}

为什么 if 语句中不能进行函数声明

因为程序执行时，程序会在本页代码之内寻找函数声明，并进行函数声明提升，而不会去查找需要条件的 if 之内的内容，

switch

将表达式的值与 case 的子表达式匹配，并执行与之匹配成功后相关联的语句。

 1n = '老虎';
 2switch (n) {
 3	// 将表达式 n 与下面的 case 后的子表达式匹配，匹配成功后会执行与其关联的代码块，
 4	// 且会执行该代码块之后的的所有代码块，为了减少代码请求，
 5	// 可以使用 break 语句终止执行该代码块之后代码块。
 6	case '老虎':
 7	case '虎':
 8	case '白虎':
 9	case '母老虎':
10		console.log('嗷嗷嗷');
11		break;
12	// 不再执行 break 之后的代码块。
13	case '猫咪':
14	case '小猫':
15	case '加菲猫':
16		console.log('喵喵喵');
17		break;

continue 和 break

continue 用于终止执行该语句之后的所有代码，并从头开始执行下一次循环。

break 用于终止本次循环，下次循环时继续执行循环体。

for

用于创建一个循环，由三个表达式和一个用于在循环中执行的代码块组成。

while

当指定表达式为 true，则循环执行一个代码块，直到表达式为 false 时结束循环。

在一个循环语句中，应当有使表达式为 false 的条件，否则会造成死循环。

do while

不管指定条件是否为 true，其代码块都会被执行一次，直到开始第二次循环时，才开始判断指定条件，直到指定条件为 false 时结束循环。

for in

用于以任意顺序循环遍历一个对象（除 Symbol 以外）的属性。

因为 for … in 是为遍历对象的属性而构建的，所以并不建议与数组一起使用； 通常与 Object 的 hasOwnPropert 方法一起使用。

 1var obj = {
 2	name: 'zhu',
 3	age: '18',
 4	gander: 'man',
 5	height: 174,
 6	__proto__: {
 7		lastName: '对象 obj 的原型 proto 上的属性',
 8	},
 9};
10Object.prototype.zxy = '原型 Object 上的属性';
11for (var key in obj) {
12	// key 为对象 obj 的属性名。
13	if (!obj.hasOwnProperty(key)) {
14		// 判断对象 obj 是否含有自身没有的属性值。
15		console.log(key + '：' + obj[key]);
16		// 返回对象 obj 自身所没有的属性名和属性值，即返回其原型上的属性。
17	}
18}

 1for (var key in obj) {
 2	if (obj.hasOwnProperty(key)) {
 3		// 判断对象 obj 是否含有自身有的属性值。
 4		console.log(key + '：' + obj[key]);
 5		// 返回对象 obj 自身拥有的属性，即不返回其原型上的属性。
 6	}
 7}
 8console.log(key + '：' + obj[key]);
 9// 返回对象 obj 的属性名与属性值（obj[key] 为对象 obj 的属性值），但因为在这里使用了 obj.key，
10// 系统会将 obj[key] 隐式转换为 obj['key']，使得 key 作为一个属性值，而不是作为一个变量。
