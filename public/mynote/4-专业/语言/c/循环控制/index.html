循环控制

某些代码被重复执行。

for

执行 1，判断 2，当 2 为 true 时则执行 A，之后再执行 3，此后一直判断 2，直到 2 为 false 时并结束循环；如果 2 为 false，则直接结束循环。

1for(1; 2; 3)
2{
3    A;
4}

其中

1 只执行一次。

3 执行完后才意味着一次 for 循环执行完毕。

3 执行完后一定会再执行一次 2，且不管表达式2的结果为 true 或 false 都会执行一次 2。

1#include <stdio.h>
2int main(void) {
3	int sum = 0;
4	for (int i = 1;i < 10;i++)
5	  printf("%d\n", i);
6	//0 - 10 的自然数。
7}

降序

浮点数的储存问题

float 和 double 类型的数据不能保证可以精确的存储某个小数，也就是你将 5.5 赋给 x，此时变量 x 中，实际储存的可能是 5.49999999999，即 x 中可能储存的是一个近似值。

for 循环嵌套

执行 1，判断 2，2 为 true，执行 4，判断 5，5 为 true，执行 A，再执行6，判断 5，5 为false，执行 3，判断 2，2 为 false，执行 B。

1#include <stdio.h>
2int main(void) {
3	for (1; 2; 3)
4	        for (4; 5; 6)
5	            A;
6	B;
7}

 1#include <stdio.h>
 2int main(void) {
 3	int i, j;
 4	for (i = 0; i < 10; i++)//该循环执行了 10 次
 5	for (j = 0; j < 10; j++)//该循环被外层循环执行了 10 次
 6	printf("内层 %d\n", j);
 7	//每次循环返回 0 - 9
 8	printf("i = %d\n", i);
 9	//最终 i 为 10
10	printf("j = %d\n", j);
11	//最终 j 为 10
12}
