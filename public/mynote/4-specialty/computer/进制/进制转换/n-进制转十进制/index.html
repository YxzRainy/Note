N 进制转十进制

给定一个 N 进制的数 25，转换为十进制为：5 + 2 * N。比如 5 进制的 32 转为十进制为：2 + 3 * 5 = 17。

十进制转十进制

1234 —> 4 * 10º + 3 * 10¹ + 2 * 10² + 1 * 10³ = 1234

二进制转十进制

10101 —> 1 * 2º + 0 * 2 ¹ + 1 * 2² + 0 *2³ + 1 * 2⁴ = 21

八进制转十进制

371 —> 1 * 8º + 7 * 8¹ + 3 * 8² = 249

十六进制转十进制

E01A —> 10 * 16º + 1 * 16¹ + 0 * 16² + 14 * 16³ = 4106
