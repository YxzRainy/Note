自增与自减#

前自增++i与后自增 i++.

异同#

相同

最终都会使 i 的值加 1.

不同

i++返回原来的值，

++i返回加 1 后的值。

#include <stdio.h>

int main()
{
    int a,b,c,d;
    a = b = 3;
    // return 3
    c = a++;
    // return 4
    d = ++b;
    printf("c = %d\nd = %d",c,d);
    return 0;
}
