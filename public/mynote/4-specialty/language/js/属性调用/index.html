属性调用

通常调用属性的格式一般为 fun.arr，实际上系统会隐式将 fun.arr 变为 fun[“arr”]，因为前者调用更加灵活且简洁。

var demo = {
	wife1: { name: 'lei' },
	wife2: { name: 'zheng' },
	wife3: { name: 'xing' },
	wife4: { name: 'zhu' },
	fun: function (num) {
		var a = ['wife' + num];
		console.log(a);
		return this['wife' + num];
		// 看不太懂
	},
};

demo.fun(1);
demo.fun(2);
demo.fun(3);
demo.fun(4);
