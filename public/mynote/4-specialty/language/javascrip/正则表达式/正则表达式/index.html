正则表达式

一种查找以及字符串替换的操作。

直接量（推荐使用）

var reg = /abcde/i;

var str = 'abCDE';

var result = reg.test(str);
console.log(result);

new RegExp

var str = 'abCDE';

var reg = new RegExp('abc', 'i');

var result = reg.test(str);

console.log(result);
