链接查询

当需要从多张表中查询数据，而这些表之间有关联字段的时候，需要使用连接查询。

笛卡尔乘积错误

表 1 有 N 行，表 2 有 M 行，最终的查询结果就是 M * N 行。

SELECT name,boyName 
FROM beauty,boys

导致的原因是因为没有添加有效的链接条件。

只需要添加有效的链接条件即可。

SELECT name,boyName 
FROM beauty,boys
WHERE beauty.boyfriend_id = boys.id;

内连接

从结果表中删除与其他被连接表中没有匹配行的所有行。

SQL 92 只支持内链接。

SQL 99 支持内链接，外连接以及交叉链接。并且该版本的语法要比 SQL 92 的可读性要高。
