Echarts 折线图

图表类型

在series下配置type.

type: 'line',

常用配置

标注图表区域

标记图表中某个范围的数据。

在series下配置markArea

markArea: {
    data: [
        [
            {
                xAxis: '李宁'
            },
            {
                xAxis: '安踏'
            }
        ],
        [
            {
                xAxis: '耐克'

            },
            {
                xAxis: '阿迪'
            }
        ],
    ],
},

线条是否平滑显示

在series下配置smooth

smooth: true,

线条样式

在series下配置lineStyle

lineStyle: {
    color: 'red',
    type: 'dashed'
}

线条区域样式

在series下配置areaStyle

areaStyle: {
    color: 'green'
}
