npm install 没反应

运行npm install时，出现以下错误

 idealTree:Test: sill idealTree buildDeps

解决办法

设置为淘宝镜像

npm config set registry https://registry.npm.taobao.org

检查是否设置成功

E:\Code\Web\Test>npm config get registry
https://registry.npmjs.org/

继续安装

npm install

安装成功！
