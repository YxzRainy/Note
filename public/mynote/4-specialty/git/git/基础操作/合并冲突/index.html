合并冲突

有时候，我们在使用git merge命令进行合并的时候，会出现以下提示。

Auto-merging Rainy.txt
CONFLICT (content): Merge conflict in Rainy.txt
Automatic merge failed; fix conflicts and then commit the result.

会发现使用git merge自动合并失败了，git 提示我们需要手动 修复冲突，然后提交结果。

如何解决冲突

我们只需要把 Git 合并失败的文件手动编辑为我们希望的内容，再git commit即可。
