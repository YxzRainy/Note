JavaScript Date

Date 创建一个 Date 实例，该实例呈现时间中的某个时刻。 Date 对象基于格林威治 Unix Time Stamp，即自 1970年 1 月 1 日（UTC）起经过的毫秒数。 Date 对象有大量的设置、获取和操作日期的方法。 但它并不含有任何属性。 Date 对象中存放的都是被构造出来的那一刻的时间，而不是随着系统时间改变而改变。
