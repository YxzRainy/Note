Vue scoped

scoped 在vue组件中，给style添加scoped属性，以表示它的样式作用于当下的模块，很好的实现了样式私有化的目的，这是一个非常好的机制。 但是为什么要慎用呢？在实际业务中我们往往会对公共组件样式做细微的调整，如果添加了scoped属性，那么样式将会变得不易修改。 使用该属性后，它会给组件添加唯一的自定义属性，使我们原本的CSS 选择器变为 CSS 属性选择器，使得该组件的样式是私有化。
