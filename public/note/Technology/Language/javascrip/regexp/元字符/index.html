元字符#

\n#

将 n 标记为一个特殊字符（将特殊字符进行转义）、或一个原义字符、或一个向后引用、或一个八进制转义符。

\w#

匹配字母、数字、下划线。等价于[A-Za-z0-9_]。

var reg = /\wcd2/g;

var str = 'bcd2';

var result = str.match(reg);
console.log(result);

\W#

匹配非字母、数字、下划线。等价于 [^A-Za-z0-9_]。

var reg = /\Wcd2/g;

var str = 'b*cd2';

var result = str.match(reg);
console.log(result);

\d#

匹配一个数字字符。等价于 [0-9]。

\D#

匹配一个非数字字符。等价于 [^0-9]。

\s#

匹配任何空白字符，包括空格符、制表符\t、回车符\r、换行符\n、垂直制表符\v、换页符\f。

等价于 [ \f\n\r\t\v]。

\S#

匹配任何非空白字符。等价于 [^ \f\n\r\t\v]。

\b#

匹配一个单词边界，也就是指单词和空格间的位置。

例如， er\b可以匹配 never 中的 ’er’，但不能匹配 verb 中的 er。

var reg = /\bname/g;

var str = 'my name is rainy';

var result = str.match(reg);
console.log(result);

\B#

匹配非单词边界。er\B能匹配 verb 中的 er，但不能匹配 never 中的 er。

var reg = /\Bname\B/g;

var str = 'mynameis rainy';

var result = str.match(reg);
console.log(result);

\t#

匹配一个制表符。等价于 \x09 和 \cI。

匹配的制表符必须是字符串中所存在的\t

var reg = /\tc/g;

var str = '\tc';

var result = str.match(reg);
console.log(result);

\u#

匹配一个用四个十六进制数字表示的 Unicode 字符。

例如， \u4e2d\u56fd的匹配项是：中国。

var reg = /\u0072\u0061\u0069\u006e\u0079\u0020\u662f\u61a8\u61a8/g;

var str = 'rainy 是憨憨';

var result = str.match(reg);
console.log(result);

Unicode 字符也可以使用区间的方式进行匹配，下面是匹配一切字符。

var reg = /[\u0000-\uffff]/g;

var str = '(?)';

var result = str.match(reg);
console.log(result);

匹配一切的字符，也可以用补集的方式：原集合 + 原集合的补集 = All

// 数字字符 + 非数字字符 = 一切字符
var reg = /[\d\D]/g;

var str = '123abc...china牛逼？';

var result = str.match(reg);
console.log(result);

.#

匹配除换行符\n和回车符\r之外的任何单个字符。相当于 [^\n\r]。

var reg = /./g;

var str = '臣世雨';

var result = str.match(reg);
console.log(result);

.不能写在[]中，若写到[]中，会匹配到 null。

var reg = /[.]/g;

var str = '臣世雨';

var result = str.match(reg);
console.log(result);

x|y#

匹配 x 或 y。

var reg = /a|b/g;

var str = 'abcabvba';

var result = str.match(reg);

console.log(result);

x(?=n)#

正向肯定预查，匹配 x 后面的字符串为 n 的匹配项。

这是一个非获取匹配，也就是说，该匹配不需要获取供以后使用。预查不消耗字符，也就是说，在一个匹配发生后，在最后一次匹配之后立即开始下一次匹配的搜索，而不是从包含预查的字符之后开始。

var str = 'abaaa';
var reg = /a(?=b)/g;
var result = str.match(reg);
console.log(result);

x(?!n)#

正向否定预查，匹配 x 后面的字符串不为 n 的匹配项。

var str = 'abaaa';
var reg = /a(?!b)/g;
var result = str.match(reg);
console.log(result);
