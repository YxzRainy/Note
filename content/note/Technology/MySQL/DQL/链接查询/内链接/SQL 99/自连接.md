# 自链接

判断同一张表中的不同字段是否相等，相等则显示，不相等则过滤。需要给同一张表取不同的别名，让系统分清同一张表中的不同字段。

### 查询员工名所对应的领导名

```SQL
SELECT e.last_name AS employeeName,m.last_name AS managerName
FROM employees AS e
INNER JOIN employees AS m ON e.manager_id = m.employee_id

```
