# 交叉链接

返回左表中的所有行，左表中的每一行与右表中的所有行组合。**交叉联接也称作笛卡尔积**

## 笛卡尔乘积错误

表 1 有 N 行，表 2 有 M 行，最终的查询结果就是 M \* N 行。\*\*\*\*

```sql
SELECT b.name,boy.boyName
FROM beauty AS b,boys AS boy
```

**笛卡尔乘积也叫交叉链接：**

```js
SELECT b.name,boy.boyName
FROM beauty AS b
CROSS JOIN boys boy
```

**导致错误的原因是因为没有添加有效的链接条件**。

只需要添加有效的链接条件即可：

```sql
SELECT b.name,boy.boyName
FROM beauty AS b
LEFT OUTER JOIN boys AS boy
# 链接条件
ON b.boyfriend_id = boy.id
```
