# git log

可以让我们查看从最近到最远所有的历史提交日志，每条日志中有对应的版本号，这个版本号可以让我们回退到指定版本号的那个版本。

## git log --pretty=oneline

每条日志都只显示一行
多屏显示控制方式空格向下翻页 b 向上翻页 q 退出

## git log --graph

用于查看分支的合并图，也叫做点线图。
