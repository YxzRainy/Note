# 操作数据库

```php
// 链接数据库
$link = mysqli_connect('localhost', 'root', '123456', 'my_shop', '3306');
if (!$link) {
    exit('数据库链接失败');
}

// 对数据库进行一次查询
$res = mysqli_query($link, 'select * from hr_goods');

// 获取查询结果中的所有字段，每个字段都是一个数组
while ($row = mysqli_fetch_array($res)) {
    echo $row[1] . '、';
}
```
