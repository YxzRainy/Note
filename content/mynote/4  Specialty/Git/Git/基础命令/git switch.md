# git switch

用于切换分支。与`git checkout`命令的区别在于，使用新的`git switch`命令，比`git checkout`要更容易理解。

## git switch -c Dev

创建并切换到新的**Dev**分支。

## git switch Dev

直接切换到名为**Dev**的分支。
