# 注入

配置的实例对象中的部分内容会被**提取到 Vue 实例中**：

- `data:{}`
- `methods: {}`

该过程称之为注入，注入的目的有两个：

## 完成数据响应式

Vue 是如何知道数据被更改的？

vue 2.0，是通过`object.defineProperty()`完成的数据响应式。该方法有很多缺陷，比如我们新增或删除`data`中的属性，此时的页面会无法正常响应。

vue 3.0，是通过 `Class proxy`。

## 绑定 this

通过 `bind()`来改变 this 的指向。
