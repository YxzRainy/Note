# 向组件传递数据

用`props:[]`定义属性，并通过`attr=‘value’`的方式将属性的`value`传给元素。

```js
var MyComp = {
	props: ['title'],
	template: `<h1>{{title}}</h1>`,
};

var app = new Vue({
	el: '#app',
	components: {
		// 当属性名与属性值一样，则该属性可简写
		MyComp,
	},
	// 传入属性 title='Vue'
	template: `<div>
    <MyComp title='Vue'></MyComp>
    </div>`,
});
Vue.config.productionTip = false;
```

在组件中，属性是只读的，不可更改的，这叫做单向数据流。

**一个组件的属性，应该避免被除了自身以外的组件所更改或控制**
