# 组件

用来实现局部（特定）功能效果的代码集合。

**与 Vue 实例的区别：**

- 无`el`配置项。
- `data` 必须是一个函数，该函数返回的对象作为数据。
- 由于没有`el`，组件的虚拟DOM 树必须定义在`template` 或 `render()`中。

### 非单文件组件

一个文件中包含 N 个组件。

### 单文件组件

一个文件中只包含 1 个组件。

## 定义组件

```js
// 定义组件
var Title = {
	template: `<h1>标题</h1>`,
};
var app = new Vue({
	el: '#app',
});
Vue.config.productionTip = false;
```
