# 插值语法

用于**解析元素包含的内容**，一般使用`{{name}}`来进行插值，其中`name`必须是JavaScrip 表达式，且可以直接读取`data`中的所有数据。

```vue
<!-- 插值语法 -->
<h1>hello！{{name}}</h1>
```
