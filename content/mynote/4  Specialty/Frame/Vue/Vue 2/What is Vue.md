# What is Vue

Vue 是一套用于**构建用户界面**的**渐进式** JavaScript框架。

## Vue 的特点

1. 采用**组件化**模式，提高代码复用率，且让代码更好维护。
2. 不在使用传统的命令式编码，在Vue中，使用**声明式**编码，让我们无需直接操作 DOM，提高开发效率。
3. . 使用**虚拟 DOM + 优秀的 Diff 算法**，尽量复用 DOM 节点。
