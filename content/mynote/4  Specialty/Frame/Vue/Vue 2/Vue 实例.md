# Vue 实例

 new  一个 Vue 实例，并将其赋给一个配置对象 ：

```vue
var vm = new Vue({})
```

该配置对象`vm`中有非常多的属性，大部分属性都是以`$`和`_`开头，这是为了避免与我们定义的属性重名。

以`$`开头的属性，都是供开发者使用的；而以`_`开头的属性，是内置属性，不供我们使用。
