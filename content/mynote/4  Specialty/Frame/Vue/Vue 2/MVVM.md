# MVVM

双向数据绑定，**数据影响视图，视图影响数据**

M：模型，指 data 中的数据。

V：视图，指模板。

VM：视图模型，指 Vue 实例对象。

data 中的所有属性，最后都出现在了 Vue 实例上。

vue 实例上的所有属性，以及 Vue 原型上的所有属性，在 Vue 模板中国都可以直接使用。
