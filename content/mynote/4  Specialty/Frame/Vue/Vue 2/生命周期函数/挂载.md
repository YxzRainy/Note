# 挂载

将生成的真实 DOM 树，插入到页面上的过程，称之为挂载。

## 挂载方式

通过`el:'CSS 选择器'`来挂载。

```js
new Vue({
	// vue 代码
}).$mount('#app');
```

通过`Vue 实例.$mount('CSS 选择器')`进行稍后挂载。

```js
// 或者
var v = Vue({
	// vue 代码
});
v.$mount('#app');
```
