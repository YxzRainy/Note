# npm list

查看当前项目安装的所有模块。

### npm list jquery

查看 jquery 模块的版本号。

### npm list -g

查看本机全局安装的模块

### npm list -g --depth 0

列出全局安装的模块 带上[--depth 0] 不深入到模块的支点 更简洁。
