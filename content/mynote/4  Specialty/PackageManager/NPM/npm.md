# npm

npm 是`Node.js`的开放式模块登记和管理系统，是`Node.js`包的标准发布平台，用于`Node.js`包的发布、传播、依赖控制。

npm 提供了命令行工具，可以方便地下载、安装、升级、删除包，也可以让你作为开发者发布并维护包。

[NPM 官网](https://www.npmjs.com/)

## 安装

安装 [Node.js](http://nodejs.cn/download/)自带 npm，所以安装Node.js
