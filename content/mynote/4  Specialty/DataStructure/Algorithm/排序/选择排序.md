# 选择排序

```c
#include <stdio.h>
void main(){

int r[] = {5,7,1,3,8,12,2,9};

for(int e = 0;e < 8;e++){
	int k = e;
	for(int i = e; i < 8;i++){
		if(r[i] < r[k]){
		    k=i;
		}
	}
		int t = r[k];
		r[k] = r[e];
		r[e] = t;
}


	for(int j = 0;j < 8;j++){
			printf("%d ",r[j]);
	}
}

```
