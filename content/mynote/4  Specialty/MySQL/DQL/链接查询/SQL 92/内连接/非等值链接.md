# 非等值链接

新建查询并运行。

```sql
CREATE TABLE job_grades
(grade_level VARCHAR(3),
 lowest_sal  int,
 highest_sal int);
```

### 查询员工的工资和工资级别

```sql
SELECT salary,grade_level
FROM employees AS 员工表, job_grades 工资等级表
WHERE 员工表.salary BETWEEN 工资等级表.lowest_sal AND 工资等级表.highest_sal
```
