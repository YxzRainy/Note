# 汉化 CodeBlocks 

将汉化文件`zh_CN.mo`放置到路径`D:\AppInstallPath\CodeBlocks\share\CodeBlocks\locale\zh_CN`（没有文件夹就自行创建）



打开 CodeBlocks，**Settings** — **Environment** — **View** — **Internationalization**(勾选)，选择**Chinese(Simplified)**
