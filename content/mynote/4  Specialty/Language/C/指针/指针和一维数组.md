# 指针和一维数组

## 一维数组名

一维数组的名字是一个指针常量，它存放的是数组一个元素的地址。

```c
#include <iostream>

int main()
{
    int a[5];
    // 打印一维数组 a 中，第一个元素的地址
    printf("%#x\n",&a[0]);
    // 打印一维数组 a 的名字
    printf("%#x\n",a);
}

```

## 下标和指针的关系

如果 p 是一个指针变量，则 p[n] 永远等价于 *（p + n）
