# ES6

## 与 JS 的关系

ES 6（ES 2015），ECMAScript 是一种标准，一种规范，同时 JavaScript 又是实现这个标准用的最多的语言。

起先被推广的名字是 ES6 。然而组委会要求 ECMAScript 必须做到每年做一次更新。由此，这个版本被更名为 ES 2015，且每年都需要更新，并命名为当前年的后缀。
