# 变量的声明方式

## 变量

变量名也叫标识符，它用于储存程序中各种类型的数据。

## var

var 可以声明局部变量与全局变量。

使用 var 声明的遍变量，无论在何处进行的声明，都将在执行任何代码之前进行预编译处理。

若用 var 声明一个变量，但没有进行赋值，则变量的默认值为 undefined。
