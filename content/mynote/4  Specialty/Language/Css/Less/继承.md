# 继承

使用 `:extend()`将混合规则集继承给指定的选择器。extend 是一个 Less 伪类

继承规则集不能带`()`。

```less
// 定义一个继承，它不能有()
.center {
	position: absolute;
	left: 50%;
	top: 50%;
	transform: translate(-50%, -50%);
}

.wrap {
	width: 300px;
	height: 300px;
	background-color: aqua;
	.inner {
        // 将 .center 中的规则集继承到 .wrap .inner{} 中
        // 使用 all
		&:extend(.center);
		&:nth-child(1) {
			width: 500px;
			height: 100px;
			background-color: pink;
		}
		&:nth-child(2) {
			width: 80px;
			height: 80px;
			background-color: rgb(210, 29, 59);
		}
	}
}

```
