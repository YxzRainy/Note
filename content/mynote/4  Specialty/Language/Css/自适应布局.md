# 自适应布局

```css
@media screen and (min-width: 720px) {
	/* 当浏览器宽度大于 value 时所应用的样式 */
}

@media screen and (max-width: value) {
	/* 当浏览器宽度小于 value 时所应用的样式*/
}
```
