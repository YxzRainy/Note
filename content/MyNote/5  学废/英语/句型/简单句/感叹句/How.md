# How

how +形容词/副词 + 主语 + 谓语

How careful he is！（他多细心啊！)

How fast he runs！（他跑得多快啊！）

how + 形容词 + 冠词(a、an) + 可数名词的单数 + 主语 + 谓语

How cute a dog it is!(它是多么可爱的一只狗啊！)
