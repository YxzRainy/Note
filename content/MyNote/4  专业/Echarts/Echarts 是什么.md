# Echarts 是什么

ECharts 是一个使用 JavaScript 实现的开源可视化库，兼容性强，

底层依赖矢量图形库 ZRender，提供直观，交互丰富，可高度个性化定制的数据可视化图表。

使用 Echarts 的时候，只需要更改配置项即可。
