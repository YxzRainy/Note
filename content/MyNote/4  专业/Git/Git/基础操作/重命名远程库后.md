# 远程库更改默认分支

我们在 Github 上更改仓库的默认分支后，或更改了分支名后，我们需要在本地库下执行以下代码

```cmd
git branch -m master Diary
git fetch origin
git branch -u origin/Diary Diary
git remote set-head origin -a
```
