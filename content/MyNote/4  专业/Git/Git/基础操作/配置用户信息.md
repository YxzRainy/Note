# 配置用户信息

安装好 Git 后，在命令行或终端中使用下面的命令可以设置 Git 自己的名字和电子邮件。这是因为Git是分布式版本控制系统，所以，每个机器都必须自报家门：你的名字和 Email 地址。

```
git config --global user.name "YxzRainy"
git config --global user.email "1@qq.com"
```

## 查看配置

配置好之后可以使用`git config -l`查看配置：

```
git config -l
```
