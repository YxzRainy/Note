# 文本溢出处理

实现单行文本的溢出显示省略号可以直接用 text-overflow:ellipsis 属性来实现，当然还需要加宽度`width`属来兼容部分浏览。

```css
overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
```
