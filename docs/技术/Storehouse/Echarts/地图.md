# 地图

## 常用配置

### 地图样式

配置 option —> geo —> itemStyle 下的`normal` .

```js
normal: {
    //面积区域颜色
    areaColor: "#008C8C",
    // 区域分界线颜色
    borderColor: "rgba(43, 196, 243, 1)",
    //区域分界线
    borderWidth: 1
},
```
