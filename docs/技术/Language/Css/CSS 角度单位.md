# CSS 角度单位

## deg

度（Degress），一个圆共 360 度。

**90deg = 100grad = 0.25 turn ≈ 1.570796326794897rad**

```css
transform: rotate(90deg);
```

## grad

梯度（Gradians），一个圆共 400 梯度。

```css
transform: rotate(2grad);
```

## rad

弧度（Radians），一个圆共 2π 弧度。

## turn

圈数（Turns），一个完整的圆为 1 圈。
