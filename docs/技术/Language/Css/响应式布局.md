#  响应式布局

## 对移动设备开启响应式

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

`name="viewport"`：调整可视窗口

`width=device-width：`设置可视窗口的宽度为当前设备的宽度

`initial-scale=1.0"`：设置初始的缩放比例
