# 安装

去官方下载[PHPStudy](https://www.xp.cn/download.html)并安装，这玩意自带 MySQL 和 Apache。

## 配置环境变量

复制 MySQL 目录下的 bin 路径，然后将这个路径添加到系统环境变量 Path 中。

我这里添加环境变量是的是：`D:\AppInstallPath\PhpStudy\phpstudy_pro\Extensions\MySQL5.7.26\bin*`
