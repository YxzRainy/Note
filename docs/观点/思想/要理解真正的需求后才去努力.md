# 要理解真正的需求后才去努力

在你努力做出东西，抛出鱼饵之前，你很难理解真正的需求。

而这世界上有很多需求，全部是被创造者自己创造出来的。

比如：

汽车发明以前，你去问人们需要什么交通工具，他们会说需要一匹更快的马。

电灯发明以前，你去问人们需要什么照明工具，他们会说需要更长更大的蜡烛。

就如同在乔布斯之前，永远没有人会告诉你，我需要一部 iPhone，而今天，我们已经完全无法离开 iPhone。
