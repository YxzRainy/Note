# italic 和 oblique

**italic**：浏览器会显示一个斜体的字体样式。

**oblique**：浏览器会显示一个倾斜的字体样式。

可以发现关键之处为斜体和倾斜。

Italic 是使用了文字本身的斜体属性，

oblique 是让没有斜体属性的文字做倾斜处理。

因为有少量的不常用字体没有斜体的属性，

如果我们使用 Italic 就会没有效果，只能 Oblique 使用才能使文字倾斜。
