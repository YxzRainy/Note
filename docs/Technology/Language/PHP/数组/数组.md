# 数组

## 索引数组

用数字作为键名的数组一般叫做索引数组。用字符串表示键的数组就是下面要介绍的关联数组。索引数组的键是整数，而且从 0 开始以此类推。

## 关联数组

数组的`key`是字符串。

## 特点

- 可以使用整数或字符串作为下标。

     - 下标为整数则为索引数组。

     - 下标都为字符串，则为关联数组。

- 不同的下标（字符串、整数）可以混合存在，则为混合数组。
- 数组中元素的顺序以放入顺序为准，与指定的下标无关。
- 特殊的下标自动转换：`false`、`true`、`null`。
