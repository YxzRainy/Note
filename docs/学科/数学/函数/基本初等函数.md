# 基本初等函数

## 基础概念

定义域：函数 $x$ 的取值范围。

对应法则：使 $x、y$ 互相对应的逻辑关系 $f$。

值域：函数值 $y$ 的范围。

具体函数：函数表达式是可知的函数，如 $f(x) = 2x$。

抽象函数：函数表达式是未知的函数，含 $f(x)、g(x)$。

反函数：$y$ 为自变量，$x$ 为因变量的函数，记为 $y = f^{-1}(x)$

令 $x=0$，得 $y$ 轴交点坐标；令 $y=0$，得 $x$ 轴交点坐标。
