# 公式

### 均值不等式

$a,b>0$，则 $\sqrt{ab} \leq \frac{a + b}{2}$

应用均值不等式

$ax + \frac{b}{x} \geq 2\sqrt{ab}$

令 $ax = \frac{b}{x}$，则 $x = \sqrt{\frac{b}{a}}$

### 两点距离公式

$\sqrt{(x_1-x_2)^2 + (y_1 - y_2)^2}$

表示点 $(x_1,y_1)$ 到点 $(x_2,y_2)$ 的距离。
