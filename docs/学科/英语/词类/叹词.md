# 叹词

用于表示感叹或发出某种情感，通常处于句子的最前面或最后面，一般与句子本身没有实际的直接联系。

请注意，叹词所表达的意思高度依赖于说话者的语气，同一个叹词，说话者的语气不同，则表达的意思也不同。

- What（天哪）
- wow、oh my god（惊讶）
- well（犹豫）
- yay（高兴）
- aww（感动）
- ouch（好痛）
- bravo（鼓励）
- yep（赞同）
- nope（反对）
- eh（确认）
- oops（后悔）

- eww（厌恶）
- psst（使注意）
- sssh（使安静）
- oh man（不耐烦）
- oh no（伤心）
- for gpd's sake、Jesus christ（愤怒）
