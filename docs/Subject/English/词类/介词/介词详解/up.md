# up

## 表示运动方向

### 运动趋势的词语 +up + 位置或地点名词

表示**向...上/沿着...**。

- He ran up the hill.
- We went up the house.

## 表示位置

### up + 位置的名词

表示**在...上面**

- They live up the street.
- The cath fish up the river.
